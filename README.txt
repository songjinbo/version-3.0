version 3改动情况说明：
16-11-27
1、在version2的基础上可以实现和树莓派的通信
2、存在以下几个问题：
（1）在树莓派没运行程序时，若点击“开始”，则对话框经常出现“应用程序无响应”，原因就是等待树莓派的连接
（2）在程序运行过程中，“开始”按钮在一直闪烁
（3）--解决。打开一次树莓派只能进行一次实验。如果数据停止传输，那么下面就得重启树莓派中的程序。这个问题会影响实验效率，要重点解决
（4）--解决。打开一次程序，当第二次点击“开始”时，不会往下执行，直接显示结束。

2017-4-26改动说明：
这个版本将TCP传输改成了UDP传输，但是由于UDP传输每次sendto的量是有限制的，所以UDP传输没有成功实施。

2017-5-5改动说明：
（1）对采用UDP协议，在应用层对传输做了处理
（2）修改了界面
（3）为了方便测试传输速率，对star.cpp文件的返回值做了修改（注意：以后需要修改回来）
（4）解决了程序运行过程中“开始”按钮一直闪烁的问题
（5）DisplayImage函数中有个语句Sleep()只是用来减缓处理速度的，并没有其他用途（以后需要删除这条语句）
